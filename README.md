# Spring Themed To Do List Website

## Description: 
This is a spring themed to do list website for you to put your day to day tasks on.

## Why We Made This Project
This project was made by me (Yuki), Alexis, and Lyrica for a hackthon we attended.

We created this project to:
- Learn and practice new skills as well as relearn some.
- To have a basic to do list that doesn't have any distractions


## How We Made This Project
This project was built using:
- Html, Javascript, and CSS
- Github, VScode, and Piskel

Process:
- We planned the general idea of what it would look like and went from there
- Wrote and tested code
- Fixed bugs and improved the look of it


## Challenges & What We Learned
Challenges:
- Faced issues with debugging and fixing errors
- LOTS of merge conflicts
- Managing time and organizing the project

What We Learned:
- ALWAYS double check your merge conflicts
- Write clear commit messages
- ALWAYS test your code before pushing it
